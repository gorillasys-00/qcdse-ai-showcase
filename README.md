# たった8日間で構築したエンタープライズ建設AI SaaS「QCDSE AI」

![QCDSE AI Logo](../frontend/public/vite.svg)

##  プロジェクト概要
「QCDSE AI」は、建設業界の若手技術者が抱える「膨大な書類作成業務」と「経験不足による判断迷い」を劇的に解消するために開発された、次世代型の施工管理サポートSaaSです。

AIエージェント（Antigravity）との完全ペアプログラミングにより、構想から本番デプロイまで、わずか**8日間**という驚異的なスピードでエンタープライズ品質のシステムを構築しました。

- **ターゲット**: 建設現場の若手エンジニア
- **コアバリュー**: 書類残業ゼロ、技術基準の即時検索、現場でのリアルタイム学習

---

##  技術スタック
モダンかつスケーラブルな構成を採用し、AIの推論能力を最大限に引き出しています。

- **Frontend**: Vite + React, Tailwind CSS (レスポンシブデザイン / Z-layout)
- **Backend**: Node.js, Express, Cloud Run
- **Database / Auth**: Supabase (PostgreSQL / pgvector)
- **AI Engine**: Google Gemini 1.5 Flash / 2.5 Pro (Hybrid Inference)
- **Infrastructure**: Firebase Hosting (Frontend), Google Cloud Run (Backend)

---

##  技術的ハイライト (Technical Moats)

### 1. 職人技レベルのA3フルカラー帳票生成
ブラウザ側（jspdf + html2canvas）で、建設業界標準のA3サイズ報告書を、1ミリのズレもなく正確にレンダリングする独自ロジックを実装。印刷時のスケーリング最適化（既定100%でのフィット）により、そのまま提出可能なクオリティを実現しました。

### 2. 厳格な施工基準に基づく「ハルシネーション・ゼロ」RAG
200万文字を超えるJASS（日本建築学会標準仕様書）や法令データをセマンティック検索。AIが「それらしい嘘」をつくのを防ぎ、必ず引用元を提示するインテリジェントな技術レポート生成機能を搭載しています。

### 3. ハイブリッドAI推論によるコスト・精度の最適化
- **通常日報**: 高速なGemini Flashでリアルタイム処理。
- **週次・月次の俯瞰分析**: 高度な推論が可能なGemini Proを採用。
文脈の重要度に応じてモデルを自律的に使い分けることで、UXとコストパフォーマンスを両立しました。

### 4. 堅牢なインフラとセキュリティ設計
Cloud Runのメモリ最適化（ストリーミングパースによるOOM回避）や、厳格なContent Security Policy (CSP) 設定、GA4/Microsoft Clarityによる徹底したアクセス解析導入など、リリース初日からエンタープライズ水準の運用基盤を構築しました。

##  開発手法: AI Agent Pair Programming
本プロジェクトの最大の特徴は、人間とAIエージェント（Antigravity）の境界線を曖昧にした**自律型ペアプログラミング**です。
人間が「目的」を定義し、AIが「実装・検証・デプロイ」を自律的に実行。エラーが発生した際も、ログ解析から修正案の提示までをAIが自律的にループさせることで、従来の3倍以上の開発効率を達成しました。
